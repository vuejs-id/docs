## Proses Development

Jika anda menggunakan NPM
``` bash
# npm
$ npm install
$ npm start # dev server di http://localhost:4000

# yarn
$ yarn
$ yarn start # dev server di http:://localhost:4000
```

Jika anda menggunakan Yarn
``` bash
$ yarn
$ yarn start # dev server at http://localhost:4000
```
